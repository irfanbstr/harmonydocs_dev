    Penjelasan Modul Queue Management

    Modul ini digunakan untuk membantu memberikan layanan pelanggan yang lebih baik bahkan pada jam sibuk dan mempersingkat antrean.

    Modul Queue Management

    Dashboard

    Untuk membuka Dashboard Queue Management user klik modul maka setelah itu akan muncul halaman seperti gambar dibawah ini:

    Gambar 25. Tampilan Dashboard Queue Management

    Menu Queue

    Bagian ini user akan menjalankan sistem antrean melalui menu. Caranya adalah klik Menu Queue > Sub Menu Counters seperti yang terlihat pada gambar berikut:

    Gambar 26. Tampilan Letak Menu Queue

    Setelah user klik pada Sub Menu Counters akan muncul daftar konter yang sudah tersedia dimana user bisa memilih dan memulai proses. Misal user ingin memulai proses pada konter pendaftaran maka user pilih Counter Pendaftaran seperti berikut:

    Gambar 27. Tampilan Tombol Start Processing

    Setelah user klik Start Processing maka akan muncul form untuk memilih department seperti berikut:

    Gambar 28. Tampilan Form Select Department

    Deskripsi Form Select Department

    Kolom Deskripsi
    Department Pilih Department *Mandatory
    Text Before Number Input Text sebelum nomor antrean disebut
    Text After Number Input Text sesudah nomor antrean disebut

    Input data-data yang harus diisi jika selesai bisa menekan tombol Start Processing untuk memulai proses dan tombol Cancel untuk membatalkan. Setelah user klik tombol Start Processing maka akan muncul tampilan yang berfungsi untuk melakukan pemanggilan nomor antrian seperti berikut:

    Gambar 29. Tampilan Pemanggil Nomor Antrean

    Menu Token Interface

    Bagian ini user akan membuka daftar Token Interface pada sistem caranya pilih Menu Token Interface > Sub Menu Token Interface dan akan muncul gambar berikut:

    Gambar 30. Tampilan Daftar Token Interface

    Untuk memulai sesi baru user bisa klik pada tombol New Session atau Resume untuk melanjutkan sesi.

    Setelah user klik tombol New Session atau Resume akan muncul antarmuka pada Kiosk seperti gambar berikut:

    Gambar 31. Tampilan Kiosk

    Jika user ingin membuat untuk pasien baru maka user klik pada tombol New Patient kemudian user akan melihat halaman baru untuk mengisi data mengenai pasien baru seperti berikut:

    Gambar 32. Tampilan Kiosk Pasien Baru

    Deskripsi Form Pasien Baru Details

    Kolom Deskripsi
    Name Input Name
    Mobile No. Input Mobile No.

    Input data-data yang harus diisi jika selesai bisa menekan tombol Generate Token untuk generate token pasien baru. User bisa klik tombol Cancel pada sebelah kiri atas untuk membatalkan.

    Sedangkan jika user ingin membuat token untuk pasien lama maka user klik pada tombol Existing Patient. Kemudian user akan melihat halaman baru untuk memilih layanan medis yang dibutuhkan pasien lama seperti gambar berikut:

    Gambar 33. Tampilan Layanan Medis untuk Pasien Lama

    Misalkan pasien lama membutuhkan layanan medis penunjang medis maka user klik pada tombol Medical Support.

    User kemudian akan melihat form untuk mengisi data pasien lama seperti berikut:

    Gambar 34. Tampilan Form Detail Penunjang Medis Pasien Lama

    Deskripsi Form Penunjang Medis Pasien Lama Details

    Kolom Deskripsi
    Pilihan Pilih EMR, NIK, Birthday untuk mencari data pasien lama
    Birthday Input Birthday pasien lama *Mandatory
    Name Input Name *Mandatory
    Mobile No. Input Mobile No. *Mandatory
    Department Pilih Department

    Input data-data yang harus diisi jika selesai bisa menekan tombol Generate Token untuk generate token pasien lama. User bisa klik tombol Cancel pada sebelah kiri atas untuk membatalkan.

    Menu Queue Display

    Bagian ini user akan menampilkan nomor antrean pada monitor melalui menu Queue Display > Sub Menu Queue Management Screens yang terlihat seperti gambar berikut:

    Gambar 35. Tampilan Letak Menu Queue Management

    Setelah user klik pada sub menu Queue Management Screens user akan melihat tampilan daftar antrean setiap departemen di Rumah Sakit seperti terlihat pada gambar berikut:

    Gambar 36. Tampilan Daftar Antrean

    Misal user memilih antrean pendaftaran maka user klik pada Antrean Pendaftaran dan akan muncul tampilan berikut:

    Gambar 37. Tampilan Antrean Pendaftaran

    Untuk menampilkan nomor antrean pada monitor user harus klik pada Queue Display URL terlebih dahulu. Setelah user klik akan muncul gambar berikut:

    Gambar 38. Tampilan Nomor Antrean Pada Monitor
